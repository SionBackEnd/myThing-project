## 🙋🏻‍♂️ Introduce
- 구매하고 싶은 물건을 나만의 위시리스트에 담아 보관할 수 있어요. 
- 내가 원하는 상대만 친구로 등록시켜 내가 원하는 상품만 공유 가능해요.
- 친구의 위시리스트 아이템을 선물하기 버튼으로 선물해줄 수 있어요.
